import processing.serial*;

Serial port;
float ball = 0;

void setup() {
 size (1000, 1000);
 port = new Serial(this,"/dev/cu.usbmodem1431", 9600);
 port.bufferUntil('\n');
}

void draw(){
 if (ball > 100 && ball < 200){   stroke(0, 250, 50);
  fill(0, 150, 200);
  strokeWeight(20);
  line(mouseX, mouseY, pmouseX, pmouseY);
 } else if (ball > 200 && ball < 500){   stroke(0, 200, 200);
  fill(0, 150, 200);
  strokeWeight(33);
  line(mouseX, mouseY, pmouseX, pmouseY);
 } else if (ball > 500 && ball < 800){   stroke(200, 0, 200);
  fill(0, 150, 200);
  strokeWeight(50);
  line(mouseX, mouseY, pmouseX, pmouseY);
 } else {
  background(255);
 }
}

void serialEvent (Serial port){
 ball = float(port.readStringUntil('\n'));
}

DOG DRAWING

with Cambria Gosh

Dog Drawing started to develop an aspect of dog training outlook that could be used in everyday day life. This project will enable animals to draw on a computer through movements of walking. The choice of animal came from the amount of movement a dog makes daily and the obedient aspect of these animals. The computer will be able to sense whether the dog is sitting or standing from pressure on the sensors. His location will be located on the screen. This experience creates a beautiful image and a dog training aspect.
